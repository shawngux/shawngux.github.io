Solidty变量类型之枚举（Enums）
Monday, Jul 23, 2018

枚举可以看成是有确定元素值的数据类型，例如，我们平常提到的方向，简单来说有东、南、西、北四个方向，那么我们可以定义一个Direction这个枚举类型。

pragma solidity ^0.4.24;

contract Test {
    enum Direction {East, South, West, North}
    
    Direction direction;
    
    function setDirection() {
        direction = Direction.East;
    }
    
    // output: 0 可看成下标
    function getDirection() constant returns (Direction) {
        return direction;
    }
}
