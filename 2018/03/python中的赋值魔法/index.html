Python中的赋值魔法
Thursday, Mar 15, 2018

在Python，用好一些赋值的小技巧可以让我们程序很简洁高效。

序列解包

也叫递归解包，即将多个值的序列解开，然后放到变量的序列中。可以一次进行多个赋值操作。

 x, y, z = 1, 2, 3
 print(x, y, z) # 返回1 2 3
 
 x, y = y, x # 两个变量值互相交换，很有用哦
 print(x, y, z) # 返回2 1 3
 
 names = 'shawn', 'tom', 'jack'
 print(names) # 返回('shawn', 'tom', 'jack') 元组
 
 a, b, c = names
 print(a + ',' + b + ',' + c) # 返回 shawn,tom,jack
 
 student = {'name': 'kate', 'age': 25}
 k, v = student.popitem()
 print('key=',k, 'value=', v) # 返回 key= age value= 25

小结：使用序列解包赋值的时候，变量的个数要和赋的值个数相同。

链式赋值

这个简单，即将同一个值赋给读个变量

 x = y = 10
 
 # 接收函数的返回值
 x = y = somefunc()

增量赋值

增量赋值也是一种简便赋值方式，即在原变量上增加某个值。例如，之前我们要在x上加1可以像 x = x + 1 这样，而使用增量赋值：x += 1 即可。对于 +、-、*、/等标准运算符都适用。

 x = 5
 x += 1
 x -= 2
 x *= 3
 x /= 4

对于其他类型如字符串：

 name = 'shawn'
 name += 'gu' # 返回 'shawngu'
 name *= 2 # 返回 'shawngushawngu'
