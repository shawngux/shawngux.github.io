Python字符串常用的方法
Thursday, Mar 8, 2018

Python为字符串的操作提供了相当丰富的方法，这里介绍一些常用的。

介绍字符方法前，简单介绍一个下字符串常量。

String constants

Python字符串模块中，定义了一些字符串常量，可直接拿来用，很方便。

  • string.ascii_letters：包含所有大写和小写字母的字符串。
import string
print(string.ascii_letters) # abcdefghijklmnopqrstuvwxyzABCDEFGHIJKLMNOPQRSTUVWXYZ
  • string.ascii_lowercase：包含所有小写字母的字符串。
print(string.ascii_lowercase) # abcdefghijklmnopqrstuvwxyz
  • string.ascii_uppercase：包含所有大写字母的字符串。
print(string.ascii_uppercase) # ABCDEFGHIJKLMNOPQRSTUVWXYZ
  • string.digits：包含0~9数字的字符串。
print(string.digits) # 0123456789
  • string.hexdigits：包含十六进制中所有字符的字符串。
print(string.hexdigits) # 0123456789abcdefABCDEF
  • string.octdigits ：包含八进制中所有数字的字符串。
print(string.octdigits) # 01234567
  • string.punctuation：包含所有的标点符号。
print(string.punctuation) # !"#$%&'()*+,-./:;<=>?@[\]^_`{|}~
  • string.printable：包含可打印字符的字符串。
print(string.printable) # 0123456789abcdefghijklmnopqrstuvwxyzABCDEFGHIJKLMNOPQRSTUVWXYZ!"#$%&'()*+,-./:;<=>?@[\]^_`{|}~

字符串方法

find

find方法可以在一个较长的字符串中寻找子串。如果存在，返回子串所在位置的最左端索引，如果不存在该子串，则返回-1。另外可指定搜索索引范围。

str = 'this is a test string for find method'
print(str.find('test')) # 返回10

print(str.find('test', 4)) # 指定从索引4开始搜索，返回10

print(str.find('test', 4, 20)) # 指定从索引4开始到索引20结束，在这个范围内搜索，返回10

print(str.find('hello')) # 不存在这个子串，返回-1

注：判断一个字符串是否在另一个字符中，可用 in。

print('el' in 'hello,world') # True

join

join方法用来将一个序列中的元素用指定的字符连接，生成一个新的字符串。

sep = '-' # 连接符
target_seq = ['1', '2', '3']
print(sep.join(target_seq)) # 1-2-3

target_seq = ('a', 'b', 'c')
print(sep.join(target_seq)) # a-b-c

split

split通过自定字符对字符串进行分割，返回字符串列表。

str = 'tom,lily,jack'
print(str.split(',')) # ['tom', 'lily', 'jack']

str = 'chinese \nmath \nenglish'
print(str.split()) # 默认空格 ['chinese', 'math', 'english']

print(str.split(' ', 1)) # ['chinese', '\nmath \nenglish'] 只分割一个

lower

lower将字符串大写字符转成小写。

str = 'ABDdeH'
print(str.lower()) # abddeh

upper

upper将字符串中的小写字符转成大写。

str = 'Welcome to China'
print(str.upper()) # WELCOME TO CHINA

title

返回标题化的字符串，意思就是将所有单词首字母转成大写

str = 'this is a title'
print(str.title()) # This Is A Title

capitalize

将字符串的第一字符转成大写。

str = 'this is a example'
print(str.capitalize()) # This is a example

replace

将某个字符串中的指定子串使用另外指定的字符串替换。

str = 'good boy, welcome to study python'
print(str.replace('boy', 'girl')) # good girl, welcome to study python

str = 'good good good day'
print(str.replace('good', 'nice', 2)) # 可以指定替换的次数 nice nice good day

strip

去除字符串两侧的空格，也可以指定去除的字符。

str = ' hello, study python happily '
print(str.strip()) # hello, study python happily

print(str.strip(' hy')) # ello, study python happil

translate

translate功能和replace类似，可以替换掉字符串某个部分。translate是通过转换表的方式来实现转换。转换表中定义的是待替换的字符和指定替换的字符的对应关系。使用string模块中的maketrans来实现。

table = str.maketrans('abcde', '12345') # 前面是待替换的字符，后面是指定替换的字符

target = 'hello, today is a nice day'
print(target.translate(table)) # h5llo, to41y is 1 ni35 41y

小结
