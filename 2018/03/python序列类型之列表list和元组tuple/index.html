Python序列类型之列表(list)和元组(tuple)
Saturday, Mar 10, 2018

列表（List）

列表是一种可变序列类型，所以通用的序列操作，如索引、分片、连接、乘等都适用于列表。除此之外，还有其他的一些改变列表的方法，如列表元素赋值、元素删除等等，下面概述一些常用操作。

字符串不能像列表一样被修改，有时候根据字符串创建列表会相当有用。如：

 list('hello') # 返回['h', 'e', 'l', 'l', 'o']

元素赋值

即可以通过索引来单个元素赋值，也可以通过分片来批量赋值。具体看代码：

 # 索引赋值修改元素
 nums = [1, 2, 3]
 nums[1] = 5
 print(nums) # 返回[1, 5, 3]
 
 names = ['Tom', 'Lucy', 'Viky']
 names[0] = 'shawn'
 print(names) # 返回['shawn', 'Lucy', 'Viky']
 
 # 分片赋值修改元素
 strs = list('shawn.gu') # 返回['s', 'h', 'a', 'w', 'n', '.', 'g', 'u']
 strs[-3:] = list('lalala') # strs返回['s', 'h', 'a', 'w', 'n', 'l', 'a', 'l', 'a', 'l', 'a'] -- 插入了新元素
 numbers = [1, 2]
 numbers[1:] = [4, 5, 6, 7, 8]
 print(numbers) # 返回[1, 4, 5, 6, 7, 8] -- 替换并且插入新元素

删除元素

可以通过del语句来删除列表元素，也可以使用分片实现删除效果，具体看代码：

 # 通过del删除元素
 names = ['Tom', 'Kate', 'Shawn', 'Viky', 'Jack']
 del names[2]
 print(names) # 返回['Tom', 'Kate', 'Viky', 'Jack']
 
 # 通过分片删除元素 -- 给一个空列表即可
 names[0:2] = []
 print(names) # 返回['Viky', 'Jack']

注：del 语句除了可以删除列表元素外，还可以用于删除字典元素，甚至还可以删除变量。

下面是列表对象的方法使用：

append

append方法用来在列表末尾追加新的元素：

 names = ['Tom', 'Kate']
 names.append('Jack')
 print(names) # 返回['Tom', 'Kate', 'Jack']

count

count用来统计某个元素在列表中出现的次数：

 nums = [1, 2, 1, 4, 5, 6, 2, 5]
 print(nums.count(1)) # 返回2
 
 lists = [[1, 2], ['shawn', 'jack'], [1, 2]]
 print(lists.count(['shawn', 'jack'])) # 返回1

extend

extend用来在列表末尾一次性追加另一个列表的多个值。即可以用新的列表来扩展原来的列表：

 nums = [1, 3, 1, 4]
 exts = [5, 2, 0]
 nums.extend(exts)
 print(nums) # 返回[1, 3, 1, 4, 5, 2, 0]

注：extend方法和+号连接列表不同，+号连接会返回一个新的列表，而extend直接在原列表后追加，修改的是原列表。

index

index用来在列表中找出某个匹配值的索引位置：

 names = ['Tom', 'Kate', 'Shawn', 'Viky', 'Jack']
 names.index('Viky') # 返回3
 
 names.index('Lily') # 报错，因为没有这个元素

insert

insert方法可用来将给定元素插入到列表指定位置，第一个参数是列表索引位置，第二个参数为给定元素：

 ages = [10, 13, 15, 21, 25, 28, 33]
 ages.insert(2, 30)
 print(ages) # 返回[10, 13, 30, 15, 21, 25, 28, 33]

pop

pop可用来移除列表中指定索引的元素，不给索引默认移除最后一个元素，并返回移除的元素：

 n = [1, 3, 5, 7, 9]
 n.pop() # 移除并返回9
 n.pop(1) # 移除并返回3

remove

remove方法用来移除列表中匹配的值：

 names = ['Tom', 'Kate', 'Shawn', 'Viky', 'Jack']
 names.remove('Jack')
 print(names) # 返回['Tom', 'Kate', 'Shawn', 'Viky']

注：remove只有一个参数，给定的参数为元素的值，这和pop不同，pop给定的是元素的索引。另外remove方法没有返回值。

reverse

reverse方法用来将列表中的元素倒置，即反向存放：

 nums = [2, 4, 6]
 nums.reverse()
 print(nums) # 返回[6, 4, 2]

sort

sort用来对列表元素进行排序。是指在原列表内部进行排序，而不是返回排序的列表副本：

 nums = [3, 2, 8, 5, 11, 0, 9, 6]
 nums.sort()
 print(nums) # 返回[0, 2, 3, 5, 6, 8, 9, 11]

sort方法返回值是None，所以通过sort方法排序后的结果进行排序是不正确的，即x=y.sort()这样是得不到正确结果的。如果得到x排序后的副本，而x本身不变得话，可以通过下面方式实现：

 # 方法一
 x = [4, 1, 3, 8, 5]
 y = x[:]
 y.sort()
 print(y) # 返回[1, 3, 4, 5, 8]
 
 # 方法二
 y = sorted(x)
 print(y) # 返回[1, 3, 4, 5, 8]

注：不能直接使用x赋值y，即y=x,这样的话x和y指向同一个列表了。

元组（Tuple）

元组也是一种序列类型，但是元组一定定义不能修改。元组通过逗号（或加上小括号）来创建，看代码：

 t = 1, 2, 3
 print(t) # 返回(1, 2, 3)，自动加上小括号()
 
 t2 = ('shawn', 'lily', 'jack')
 print(t2) # 返回('shawn', 'lily', 'jack')
 
 # 当元组中只有一个元素时，后面的逗号,一定不能省略，否则会自动转换成整数、字符串或其他类型
 t3 = (5,)
 t4 = 'shawn',

元组可以通过tuple()方法加一个序列作为参数，转成元组，这个和list一样：

 t = tuple([1, 3, 5])
 print(t) # 返回(1, 3, 5)
 
 t2 = tuple('shawn')
 print(t2) # 返回('s', 'h', 'a', 'w', 'n')
 
 t3 = tuple(('tom', 'lily', 'jack'))
 print(t3) # 返回('tom', 'lily', 'jack')

元组的应用场景有：

  • 可以作为映射和集合中当做键来使用
  • 作为函数的返回值存在。

总结

列表list和元组，尤其列表是Python经常使用到的序列类型，列表是可变的序列，可以进行赋值、删除等操作；而元组是不可变的序列，不可变的含义是指向不可变。
