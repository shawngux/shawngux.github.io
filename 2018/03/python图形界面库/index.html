Python图形界面库
Tuesday, Mar 27, 2018

Python支持多种第三方的图形界面库。常见有：

  • Pyqt：anaconda中自带

  • wxPython

推荐使用anaconda集成包作为Python环境，在anaconda中安装wxpython很简单：

conda install -c melund wxpython
  • Tkinter：这个包Python中自带

使用哪个库导入即可

# 使用pyqt
import pyqt

# 使用tkinter
import tkinter # from tkinter

# 使用wxpython
import wx # wxpython
