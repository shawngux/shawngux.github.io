Python基本语法之普通数据类型
Tuesday, Mar 6, 2018

不同的数据代表的不同的类型，那么在计算机语言中，使用不同的类型来定义不同的数据。

在解释类型之前，先说一下变量：

在python中，变量即某个值（或者引用）的名字，例如，num = 5，这时num这个名字就代表5。num就是一个变量，变量赋值（=）后，就可以在表达式中使用。

整型

即整数，如1、2、5……，这些都是整数，整数常用的操作有+、-、*、/、%等

a = 5
b = 20
print(a + b) # 25
print(a - b) # -15
print(a * b) # 100
print(a / b) # 0.25
print(a % b) # 5

浮点型

即浮点数、小数，如1.1、2.1、3.2、1.23e4等，浮点数和整数一样，有+、-、*、/、%等操作，结果还是浮点数

a = 1.2
b = 2.5
print(a + b) # 3.7
print(a - b) # -1.3
print(a * b) # 3.0
print(a / b) # 0.48
print(a % b) # 1.2

字符串

不过在哪种计算机语言中，字符串都是使用的非常多的一种类型。

  • 字符串一般使用单引号或者双引号来定义，例如
str = 'hello,world'
str = "hello,world"

当需要引号本身作为一个字符存在时，可以使用单双引号组合，或使用转义字符来转义引号

name = "I'm shawn.gu"
print(name) # I'm shawn.gu
name = 'I\'m shawn.gu'
print(name) # I'm shawn.gu

  • 拼接字符串

    • 当两个字符紧接着相连时，Python编译器会自动将其拼接成一个字符串
    grate = 'hello,' 'world'
print(grate) # 这用机制用的不多，但有时挺好用的

    注：不可使用两个变量紧接相连来拼接一个字符串，否则报错

    • 使用“+”来连接，这种方式普遍通用些
    first_name = 'shawn.'
last_name = 'gu'
full_name = first_name + last_name
print(full_name) # shawn.gu

  • 多行字符串

可以使用三个引号代替普通引号来表示长字符串， 用来跨行

str = '''this is
a very long
string'''
  • 原始字符串

原始字符串以r开头，它对于反斜线不会特殊对待。在写正则表达式时是非常有用的。

name = r'I\'m shawn.gu'
print(name) # I\'m shawn.gu

注：原始字符串末尾不能是反斜杠，否则报错，如果希望展示出来的结果是个反斜杠，可以拼接转义

str = r'Let\'s go there,' '\\'
print(str) # Let\'s go there,\
  • 字符串格式化

在Python中，字符串格式化使用的和c语言一致，用%来表示。在%的左侧放置一个字符串（待格式化字符串），右侧放置希望被格式化的值。可以使用单个值，如一个字符串或者数字，也可以使用多个值的元组来表示

format = 'hello,%s' % 'world'
print(format) # hello,world

multi_val = 'hello, %s, welcome to %s' % ('shawn', 'china')
print(multi_val) # hello, shawn, welcome to china

注：%s称为转换说明符，标记了需要插入转换值的位置。常用的转换说明符有：

符号 含义
%d 整数
%f 浮点数
%s 字符串
%x 十六进制数

小结转换时可使用的项：

  1. %字符：标记转换说明符的开始。
  2. 转换标志（可选）：- 表示左对齐；+ 表示在转换值之前要加上正负号；“” 表示正数之前保留空格；0 表示转换值不够则用0填充
  3. 最小字段宽度（可选）：转换后的字符串至少应该具有这个值指定的宽度。如果是 * 的话，宽度从元组中读取给定的值
  4. 点（.）后跟精度值（可选）：如果转换的是实数，精度值表示出现在小数点后的位数，如果转换的是字符串，则表示最大字段宽度。如果是 * ，精度取元组中给定的值。
from math import pi
'%10f' % pi # '  3.141593'  字段宽度为10
'%10.2f' % pi # '      3.14'  字段宽度为10，精度（小数位数）为2
'%.2f' % pi # '3.14' 精度为2
'%010.2f' % pi # '0000003.14' 宽度为10，不够前面使用0填充，精度为2
'%.5' % 'hello world' # 'hello' 宽度为5
'%.*s' % (3, 'hello, world') # 'hel' 元组中指定的3位
'%-10.2f' % pi # '3.14      ' 左对齐
'%+5d' % 10 # '  +10' 5位，前面加上+号
'%+2d' % -20 # '-10' 前面加上-号

布尔类型

返回值有 True、False，支持的运算有 and、or、not以及一些比较运算。

运算 逻辑表达式 描述
and x and y “与”，如果x为False，返回False，否则返回y执行的结果（两个都为True）
or x or y ”或”，如果x为False，返回y的结果值，否则返回x的结果值False（其中一个为True）
not not x “非”，如果x为True，返回False；如果x为False，返回True 
x = True
y = False

if x and y:
    print('x和y都为True')
else:
    print('x和y不全是True') # 返回 x和y不全是True
    
if x or y:
    print('x和y其中有一个为True') # 返回
else:
    print('x和y中没有任何一个为True')  
    
print(not x) # False

其他一些比较运算参考：https://docs.python.org/3/library/stdtypes.html#boolean-operations-and-or-not
