Python中的一些迭代工具
Wednesday, Mar 14, 2018

在Python中迭代序列或者其他可迭代对象时，有一些函数非常好用。

并行迭代

即程序可以同时迭代两个序列，如：

 students = ['jack', 'kate', 'tom', 'mate']
 ages = [21, 18, 23, 27]
 # 名字对应年龄
 # range为列表生成式，在这里生成从0到len(students)的整数列表
 for i in range(len(students)):
     print(students[i], '=', ages[i]) # 打印 jack = 21 kate = 18 tom = 23 mate = 27

使用内建函数zip将两个序列压缩在一起，返回一个元组为成员的列表，然后进行并行迭代。

 students = ['jack', 'kate', 'tom', 'mate']
 ages = [21, 18, 23, 27]
 # zip(students, ages) 在迭代时会返回[('jack', 21), ('kate', 18), ('tom', 23), ('mate', 27)]
 for stu, age in zip(students, ages):
     print('stu=',stu, ',age=', age)
     
 '''返回
 stu= jack ,age= 21
 stu= kate ,age= 18
 stu= tom ,age= 23
 stu= mate ,age= 27
 '''

按索引迭代

有时候想要迭代当前序列中的对象，同时还要获取当前对象的索引，看下面代码：

 # 当序列对象为字符串时
 name = 'shawn'
 for s in name:
     index = name.index(s)
     print('s=',s,',index=', index)
     '''
     s= s ,index= 0
     s= h ,index= 1
     s= a ,index= 2
     s= w ,index= 3
     s= n ,index= 4
     '''
     
 # 序列为列表
 names = ['jack', 'kate', 'tom'] 
 for name in names:
     index = names.index(name)
     print(index)

使用python内建函数enumerate。

 # 列表
 names = ['jack', 'kate', 'tom']
 for index, name in enumerate(names):
     print('index=',index,',name=',name)

小结：使用内建函数enumerate,可以很方便的迭代出序列对象的索引。

反转和排序迭代

当我们需要将一个可迭代对象进行反转或者排序后返回，这个时候reversed和sorted函数就非常有用了，reversed函数返回的是一个对象。看代码：

 greet = 'hello,world'
 print(list(reversed(greet))) # 返回['d', 'l', 'r', 'o', 'w', ',', 'o', 'l', 'l', 'e', 'h']
 
 numbers = [1,5, 3, 9, 4, 6]
 print(sorted(numbers)) # 返回[1, 3, 4, 5, 6, 9]

总结

上面介绍了一些迭代时可能用到的工具函数，使用时注意一些函数返回的是一个对象，索引或其他操作时需要进行转换，例如通过list()转换成列表等。
